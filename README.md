objectName，控件对象名称
geometry，相对坐标系。
sizePolicy，控件大小策略。
minimumSize，最小宽度、高度
maximumSize，最大宽度、高度。如果想让窗口或控件的大小固定，则可以将 minimumSize 和 maximumSize 这两个属性设置成一样的数值。
font，字体。
cursor，光标。
windowTitle，窗口标题。
windowsIcon/icon，窗口图标/控件图标。
iconSize，图标大小。
toolTip，提示信息。
statusTip，任务栏提示信息 。
text，控件文本。
shortcut，快捷键。
